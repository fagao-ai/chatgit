## 🚀 快速开始

### 前置要求

- Python 3.12+
- Node 20+
- Pnpm 10
- OpenAI API Key
- GitHub Personal Token（可选）

### 三步上手

```bash
# 1. 克隆仓库
git clone https://github.com/fagao-ai/chatgit.git

# 2. 安装依赖
uv sync

# 3. 配置环境(可选,web页面也可配置)
cp .env.example .env
# 编辑.env文件填入您的API密钥
```

### Docker 部署

```bash
docker build -t chatgit:0.0.1 .

docker run -d -p 80:80 chatgit:0.0.1
```
