## 🚀 快速开始

### 前置要求

- Python 3.8+
- OpenAI API Key
- GitHub Personal Token（可选）

### 三步上手

```bash
# 1. 克隆仓库
git clone https://github.com/yourusername/chatgit.git

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置环境
cp .env.example .env
# 编辑.env文件填入您的API密钥
```
